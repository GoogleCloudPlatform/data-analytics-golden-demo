Configuration

Instructions to configure Looker
  1. Clone this report: click here
  2. Click the 3 dots in the top right and select "Make a copy"
  3. Click "Copy Report"
  4. Click "Resouce" menu then "Manage added data sources"
  5. Click "Edit" under Actions title
  6. Click "${project_id}" (or enter the Project Id) under Project title
  7. Click "rideshare_lakehouse_curated" under Dataset title
  8. Click "looker_high_value_rides" under Table title
  9. Click "Reconnect"
  10. Click "Apply" - there should be no field changes
  11. Click "Done" - in top right
  12. Click "Close" - in top right
  13. Check report - it should have data if you have run a prediction
  14. Go to File menu | Embed report (enable embedding)
  15. Click Embed URL
  16. Copy the link (e.g. https://lookerstudio.google.com/embed/reporting/e5f638c8-4440-4a3c-be4c-38f95f1a285e/page/wmQFD)
  17. Paste the link below
  18. Click Submit
